 🚀 NASA Explorer App (Android)

NASA Explorer** is an Android application built in **Kotlin** that allows users to explore the cosmos using NASA's public APIs.
This app provides astronomy images, Mars Rover photos, and NASA image search functionality with a clean, modern, and interactive interface.  

📱 Features

- Astronomy Picture of the Day (APOD) 
  View stunning images from NASA every day with descriptions.  

- Mars Rover Photos
  Browse photos captured by Mars rovers during exploration missions.  

- NASA Image Search 
  Search through NASA's image archive to discover space-related images.  

- About Section 
  Learn about the app, its purpose, and the development team.  

- Offline Caching  
  Images and data are cached locally for offline access.  

 🛠 Technologies Used

- Language: Kotlin  
- Android Framework: Android SDK, Jetpack Compose  
- Networking: Retrofit for API calls  
- Data Storage: Room Database for offline caching, MVVM  
- UI Components: Material Design, CardView, Buttons, ScrollView  
- APIs: [NASA Public APIs](https://api.nasa.gov/)
